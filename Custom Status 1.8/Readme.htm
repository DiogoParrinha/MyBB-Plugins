Description

Custom Status

This plugin allows certain user groups to set a custom status from the user cp (or index) which is shown on the index page, profile and posts.

Install

Upload contents of the Upload folder to the root of your MyBB installation.

Go to Admin CP -> Configuration -> Plugins and Activate "Custom Status"

Then go to Settings -> Custom Status and change anything you need.

 

You can use {$post['custom_status']} anywhere in the postbit and postbit_classic templates - IT IS NOT EDITED BY DEFAULT.

Upgrade

From 1.7 to 1.8:

  • Upload contents of the Upload folder to the root of your MyBB installation.
