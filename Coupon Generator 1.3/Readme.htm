Description

Coupon Generator

Adds a new page to your forums.

This page contains a button and whenever users click it, it generates a coupon code which is sent to the Administrator's inbox as

well as to the user's inbox.

Generated Coupon codes are saved in the database and can be accessed from the Admin CP -> Tools & Maintenance -> Coupon Codes.

Comes with NewPoints compatibility, you can set a price users must pay each time they generate a coupon code.

Install

Upload contents of the Upload folder to the root of your MyBB installation.

Go to Admin CP -> Configuration -> Plugins and Activate "Coupon Generator"

Then go to Settings -> Coupon Generator and change anything you need.

Upgrade

From 1.1/1.2 to 1.3:

  • Upload contents of the Upload folder to the root of your MyBB installation, overwriting the existing content.
