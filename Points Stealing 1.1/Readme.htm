Description

Points Stealing

Allows users to steal points from other users.

A price can be set for stealing, with a chance of success and the possibility of receiving PM alerts on success/failure.

Moreover, a flood check can be enabled as well as a maximum amount of points to be stolen.

This plugin can integrate with the Shop, allowing admins to set an item as a Blocker Item - if a victim holds this item, a stealing attempt is blocked.

Install

Upload the contents of the "Upload" folder to the root of your forum.

From ACP -> NewPoints -> Plugins -> Install and Activate "Points Stealing".

From ACP -> NewPoints -> Settings -> "Points Stealing" -> Configure the plugin settings.


Upgrade

Upload the contents of the "Upload" folder to the root of your forum, overwriting all files.

Instructions

The Points Stealing page in yoursite.com/newpoints.php?action=stealing.
