Description

Attachment Downloads Log

Logs who downloads attachments. The name and id of the attachments are logged as well and the date of the download too.

You can choose which forums and groups are not logged.

Logs can be deleted manually and pruned.

 

Adds a section to the ModCP from which moderators can view all logs and filter them (it is not possible to delete or prune from ModCP).

 

Instructions

You can view the logs by going to Admin CP -> Tools & Maintenance -> Attachment Log.

Alternative, you can browse the logs from the ModCP - these allow for better management (sorting; filtering; searching).

Install

Upload the contents of the "Upload" folder to the root of your forum.

From ACP -> Plugins -> Install and Activate "Attachment Downloads Log".

Then go to Settings -> Attachment Downloads Log and change anything you need.

Upgrade

From 2.0 to 2.1:

To upgrade just overwrite the files and deactivate and activate back from the AdminCP.

From 1.1 to 2.0/2.1:

To upgrade just overwrite the files and deactivate and activate back from the AdminCP.

After that, upload the upgrade.php file to the MyBB root and run it from your browser.

Delete the upgrade file after that.
